== README

Welcome to my Rails Party Planner app. In this app you and your friends can create accounts and make parties and invite eachother. It's a great way to keep track of all the food and friends at your parties.

* Download from GitHub
* `cd` to directory in terminal
* Enter `bundle install` in terminal to install dependencies/gems
* Enter `rails s` to start the app
* Navigate to `http://localhost:3000` to begin


Built on Rails 4.2.10 and Ruby 2.3.1p112
